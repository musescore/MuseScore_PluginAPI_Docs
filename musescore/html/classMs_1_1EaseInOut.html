Detailed Description

Specialized transfer curve using an underlying Bezier curve.

The second and third control points of the Bezier curve are moveable along the x axis according to two parameters 'EaseIn' and 'EaseOut', which results in a transfer curve that goes from a straight line to an S shaped curve.
