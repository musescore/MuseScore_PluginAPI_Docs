bool Ms::OpenFileLocation::openFileLocation ( const QString &  path)
static

reveals the file in explorer (Windows) / finder (MacOs) or the default file manager in other cases.

Note that opening the default file manager doesn't show the file, but simply opens the containing folder. Note also that all platforms will fallback to opening the containing folder if they can't show the file.

Returns true on success, false on failure. See MetaEditDialog::openFileLocation example usage
