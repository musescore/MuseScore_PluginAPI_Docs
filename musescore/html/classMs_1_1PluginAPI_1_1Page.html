int Ms::PluginAPI::Page::pagenumber
read

Page number, counting from 0.

Number of this page in the score counting from 0, i.e. for the first page its pagenumber value will be equal to 0. User-visible page number can be calculated as

page.pagenumber + 1 + score.pageNumberOffset

where score is the relevant Score object.

MuseScore 3.5
