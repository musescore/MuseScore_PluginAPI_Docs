41 // @@ Cursor
42 /// Cursor can be used by plugins to manipulate the score.
43 /// Cursor object for a score can be obtained with
44 /// \ref Score.newCursor method. After creating a cursor
45 /// it does not point to any location in a score. To define
46 /// its initial location use \ref rewind or \ref rewindToTick
47 /// methods. Alternatively, you can set its
48 /// \ref inputStateMode to \ref INPUT_STATE_SYNC_WITH_SCORE "Cursor.INPUT_STATE_SYNC_WITH_SCORE"
49 /// to make cursor location be synchronized with
50 /// user-visible note input state.
51 //---------------------------------------------------------
