191 // collect lyric numbering information and determine order
192 //
193 // MusicXML lyrics may contain name and number attributes,
194 // plus position information (typically default-y).
195 // Name and number are simply tokens with no specified usage.
196 // Default-y cannot easily be used to determine the lyrics
197 // line, as it tends to differ per system depending on the
198 // actual notes present.
199 //

201 // within a MusicXML part and assigning lyrics position
202 // based on alphabetically sorting works well for all
203 // common MusicXML files.
