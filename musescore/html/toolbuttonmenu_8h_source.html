20 // ToolButtonMenu
21 // ==============
22 // This creates a button with an arrow next to it. Clicking the button triggers the default
23 // action, while pressing the arrow brings up a menu of alternative actions. Selecting an
24 // alternative action triggers it and also has some effect on the default action.
25 //
26 // The menu may contain:
27 // - Any number of alternative actions in addition to the default action.
28 // - Other actions that do not affect the default action.
29 // - These are not considered "alternative actions" for our purposes.
30 //
31 // The effect of selecting an alternative action is determined by the swapAction boolean.
32 // TRUE: The selected action replaces the default action (it becomes the new default).
33 // FALSE: The default action remains unchanged, but its icon is updated to match the
34 // icon of the selected alternative action.
35 //---------------------------------------------------------
